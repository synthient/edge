# Synthient Edge

Edge is a lightweight IP context service built on ASP.NET (.NET 10). It ingests proxy events from a Redis pub/sub feed, stores them in configurable retention buckets, and exposes enriched IP lookups over a REST API.

Intended for on-premise deployment by clients with access to the [Synthient firehose](https://docs.synthient.com/enterprise/firehose). See the [wiki](https://github.com/synthient/edge/wiki) for configuration and API reference.

---

Edge is currently in early alpha. Features and APIs are under active development. For bugs or questions, please open an issue or reach out through official channels.
